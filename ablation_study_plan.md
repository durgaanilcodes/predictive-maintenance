# Ablation Study Plan

## Objective
To compare machine failure prediction performance using:

1. Internal Sensor Features Only
2. Internal Sensor Features + External Contextual Features

## Internal Sensor Features
- Air temperature [K]
- Process temperature [K]
- Rotational speed [rpm]
- Torque [Nm]
- Tool wear [min]

## Target Variable
- Machine failure

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score

## Expected Outcome
Determine whether contextual features improve prediction performance.
