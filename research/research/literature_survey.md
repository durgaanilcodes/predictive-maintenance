# Literature Survey on Predictive Maintenance

## Paper 1: Predictive Maintenance Using Machine Learning

Objective:
To predict equipment failures before breakdown.

Method:
Machine learning algorithms trained on sensor data.

Result:
Reduced downtime and maintenance costs.

## Paper 2: IoT-Based Predictive Maintenance

Objective:
Monitor industrial equipment using IoT sensors.

Method:
Real-time sensor data collection and analysis.

Result:
Early fault detection and improved reliability.

## Paper 3: Predictive Maintenance in Manufacturing

Objective:
Improve machine reliability and efficiency.

Method:
Analysis of vibration and temperature sensor data.

Result:
Increased machine uptime and reduced failures.

## Conclusion

Predictive maintenance combines IoT sensors and machine learning to detect failures early, reduce costs, and improve operational efficiency.
