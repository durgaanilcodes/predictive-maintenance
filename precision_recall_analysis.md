# Precision-Recall Analysis

## Precision

Precision measures how many predicted failures are actually failures.

Formula:

Precision = TP / (TP + FP)

## Recall

Recall measures how many actual failures were correctly detected.

Formula:

Recall = TP / (TP + FN)

## Importance

High Precision reduces false alarms.

High Recall reduces missed failures.

## Project Objective

Our predictive maintenance system aims to achieve a good balance between Precision and Recall to detect machine failures accurately.
