# Contextual Features Analysis

## Internal Sensor Features
- Air temperature [K]
- Process temperature [K]
- Rotational speed [rpm]
- Torque [Nm]
- Tool wear [min]

## Contextual Features
- Ambient_Temperature
- Load_Density

## Engineered Features
- Temp_Difference
- Power_Index

## Purpose

The contextual features provide environmental and operational information that may improve machine failure prediction.

## Future Ablation Study

Experiment 1:
Internal sensor features only.

Experiment 2:
Internal sensor features + contextual features.

Performance metrics such as Accuracy, Precision, Recall, and F1 Score will be compared.
