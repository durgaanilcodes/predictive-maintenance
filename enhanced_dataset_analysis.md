# Enhanced Dataset Analysis

## Dataset Information

- Dataset Name: Enhanced Predictive Maintenance Dataset
- Number of Records: 10000
- Number of Features: 18

## Internal Sensor Features
- Air Temperature [K]
- Process Temperature [K]
- Rotational Speed [rpm]
- Torque [Nm]
- Tool Wear [min]

## Contextual Features
- Ambient_Temperature
- Load_Density

## Engineered Features
- Temp_Difference
- Power_Index

## Objective

The enhanced dataset combines machine sensor data with contextual and engineered features to improve machine failure prediction.

## Future Scope

The dataset will be used for ablation studies to compare:
1. Internal sensor features only.
2. Internal sensor + contextual features.
